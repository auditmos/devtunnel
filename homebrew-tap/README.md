# Homebrew Tap for DevTunnel

## Install

```bash
brew tap auditmos/devtunnel
brew install devtunnel
```

## Update Formula

After release, update `Formula/devtunnel.rb`:
1. Update `version`
2. Get SHA256 from `checksums.txt` in release
3. Replace `PLACEHOLDER_SHA256_*` with actual checksums

## Publish

Push this directory to `github.com/auditmos/homebrew-devtunnel`
